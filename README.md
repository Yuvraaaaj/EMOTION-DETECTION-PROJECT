# 🎭 Emotion Detection Project

## 📌 Overview

This project detects human emotions from facial expressions using **Deep Learning** and **Computer Vision**.
It can analyze emotions in real-time using a webcam or from input images.

---

## 🚀 Features

* 🎥 Real-time emotion detection using webcam
* 🖼️ Emotion prediction from images
* 📊 Displays confidence score for predictions
* 😄 Emoji-based output for better visualization
* 👥 Supports detection of multiple faces

---

## 🛠️ Tech Stack

* **Python**
* **OpenCV**
* **TensorFlow / Keras**
* **NumPy**

## 🎯 How It Works

1. Captures image/video from webcam
2. Detects faces using OpenCV
3. Passes face data to trained model
4. Predicts emotion (Happy, Sad, Angry, etc.)
5. Displays result with confidence score
